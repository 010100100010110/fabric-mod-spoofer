# 📦 Готовый Скомпилированный JAR

Если у вас возникли проблемы с компиляцией, используйте готовый JAR файл.

## ✅ Как использовать:

1. Скачайте файл `mod-spoofer-1.0.0.jar` из этой папки
2. Поместите его в папку `mods/` вашего Minecraft клиента
3. Запустите Minecraft с Fabric Loader

## 📍 Где находится папка mods?

**Windows:**
```
C:\Users\[ВашИмя]\AppData\Roaming\.minecraft\mods
```

**Linux:**
```
~/.minecraft/mods
```

**Mac:**
```
~/Library/Application Support/minecraft/mods
```

## ✨ Готово!

Мод будет активен при подключении к серверу с проверкой модов.

Логи будут в: `mods/.modspoofer/`
